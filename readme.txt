Steav-Ticket - Full-Stack Project
=================================

This repository contains the full-stack implementation for Steav-Ticket. 
The project is organized into two main directories: the Frontend and the Backend.

Project Structure:
------------------
1. Steav-Ticket (Frontend)
   - Framework: Vue 3 with Vite
   - Language: TypeScript
   - Testing: Vitest (Unit), Playwright (End-to-End)
   - Setup Instructions:
     cd Steav-Ticket
     npm install
     npm run dev

2. Backend (Backend API)
   - Framework: NestJS
   - Language: TypeScript
   - Setup Instructions:
     cd Backend
     npm install
     npm run start:dev

Additional Information:
-----------------------
For more detailed instructions, configuration options, and deployment guides, 
please refer to the individual README.md files located within the 'Backend' 
and 'Steav-Ticket' directories respectively.
