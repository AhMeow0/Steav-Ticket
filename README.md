# Steav-Ticket Full-Stack Project

This repository contains the full-stack implementation for the **Steav-Ticket** project. The workspace is divided into two primary directories, separating the frontend and backend architectures.

## Project Structure

### 1. Frontend (`Steav-Ticket`)
The client-side application is a modern web application built using Vue 3 and Vite.
- **Framework:** Vue 3
- **Build Tool:** Vite
- **Language:** TypeScript
- **Testing:** 
  - Unit Tests: Vitest
  - End-to-End Tests: Playwright

**Quick Start:**
```bash
cd Steav-Ticket
npm install
npm run dev
```

### 2. Backend (`Backend`)
The server-side application is a robust API built with NestJS.
- **Framework:** NestJS
- **Language:** TypeScript
- **Runtime:** Node.js

**Quick Start:**
```bash
cd Backend
npm install
npm run start:dev
```
